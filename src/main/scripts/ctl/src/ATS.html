Overview

This executable test suite is based on the following OGC specifications. Please find WCS 2.0 Range Subsetting Extension ATSs in the corresponding documents.

  • OGC Web Services Common Standard (OGC 06-121r9)
  • OGC® WCS 2.0 Interface Standard - Core (OGC 09-110r3)
  • OGC® WCS 2.0 Interface Standard - Core: Corrigendum (09-110r4)
  • OGC® GML Application Schema - Coverages (09-146r2)
  • OGC® Web Coverage Service 2.0 Interface Standard - KVP Protocol Binding Extension - Corrigendum (OGC 09-147r3)
  • OGC® Web Coverage Service 2.0 Interface Standard - XML/POST Protocol Binding Extension (OGC 09-148r1)
  • OGC® Web Coverage Service 2.0 Interface Standard - XML/SOAP Protocol Binding Extension (OGC 09-149r1)
  • OGC® Web Coverage Service 2.0 Interface Standard - Range Subsetting Extension (OGC 12-040)
  • Compliance Test Language (CTL) 0.5
